# personalhub

Personal profile card and vibe-coding playground.

## Content maintenance guide

This project keeps the editable content in `index.html` so future updates can stay lightweight.

- **Profile copy:** Update the hero section near the top of `index.html` to change the intro headline, supporting paragraph, and quick-facts pills.
- **Links:** Edit the cards in the `#links` section of `index.html` for primary external destinations such as social profiles, repositories, or newsletters.
- **Projects:** Edit the cards in the `#projects` section of `index.html` for featured work, summaries, and project URLs.
- **Misc items:** Edit the cards in the `#misc` section of `index.html` for availability notes, current focus items, or other lightweight updates.
- **Footer/contact info:** Update the `#footer` links in `index.html` for email, feeds, or any other contact destinations.

If you replace placeholder URLs or metadata, also update the `<head>` block in `index.html` so the page title, description, canonical Open Graph URL, and share image stay in sync with the visible content.
