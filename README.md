# personalhub

`personalhub` is intended to be deployed as a GitHub Pages site from the repository root using `index.html`.

## What the page contains

The landing page currently includes:

- a semantic `header` with top-level navigation anchors
- a hero section with a site name, short self-intro, site-purpose copy, and calls to action
- a main links section for primary destinations
- a featured projects section with starter project cards
- a misc / extra section for flexible supporting content
- an about section with background information
- a semantic `footer`

This structure provides a clean GitHub Pages entry point that can be customized with real links, project details, and personal content over time.
